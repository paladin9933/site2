/site2/sofar
 Приложение для получения данных

файл с моделями содержит: 

	объект posts где заданы основные поля  Заголовок, Автор, Просмотры, Позиция 
	ID(создается автоматически при миграции);

	Объект modelsEncorer расшифоровывает данные при вызове и возвращает в виде {id: Индентификатор, title: Заголовок, Avtor: Автор, position: Позиция, views : Просмотры}

При запуске окрывается простая форма в поле нужно ввести id, и нажать кнопку, 
	После идет обращение к функции: 
		postsofar из файла  views.py, 
			функция создаёт запрос в адресную строку в виде /sofarid?ID=(введеное число)&name=tp/ данный запрос получает функция re_path из файла urls.py в корне проека, и 
			вызывает функцию: 
				getdatasofar из файла views.py которая возвращает данные полученные из базы по id в виде строки в браузере, параметр name не используется планировал сделать разные типы.
